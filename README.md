# To-Do App ✅

Une application simple de gestion de tâche, développée pour m'entrainer avec React.js.📃🧾📝📋

## 🔧 Foction principales

- Ajouter une tâche
- Marquer comme faite
- Suprimer une tâche
- Interface responsive

## 🛠 Technologies utilisées

- React
- HTML / CSS
- JavaScript
- Bootstrap

## 🚀 Lancer l'application

```bash
npm install
npm run start
```
